# Design Plan - Legal Pages Layout

## Concept général

Pages légales minimalistes et élégantes, alignées sur l'identité de marque luxe de BK Agencements.

## Structure de mise en page

### Container principal
- **Largeur maximale** : 800px
- **Marge latérale** : Auto (centré)
- **Padding horizontal** : 2rem (mobile) / 4rem (desktop)
- **Padding vertical** : 4rem (mobile) / 6rem (desktop)

### Alignement
- **Texte** : Aligné à gauche
- **Largeur de ligne** : 60-70 caractères maximum pour une lecture optimale
- **Pas de justification** : Texte aligné à gauche uniquement

## Typographie

### Titres principaux (H1)
- **Police** : Bodoni Moda (serif)
- **Taille** : 2.5rem (mobile) / 3rem (desktop)
- **Poids** : 400 (regular)
- **Espacement** : Lettre-spacing normal
- **Marge inférieure** : 2rem
- **Couleur** : Noir (#000000)

### Sous-titres (H2)
- **Police** : Bodoni Moda (serif)
- **Taille** : 1.75rem (mobile) / 2rem (desktop)
- **Poids** : 400
- **Marge supérieure** : 3rem
- **Marge inférieure** : 1.5rem
- **Couleur** : Noir (#000000)

### Sous-sections (H3)
- **Police** : Raleway (sans-serif)
- **Taille** : 1.25rem (mobile) / 1.5rem (desktop)
- **Poids** : 600 (semi-bold)
- **Marge supérieure** : 2rem
- **Marge inférieure** : 1rem
- **Couleur** : Noir (#000000)

### Corps de texte
- **Police** : Raleway (sans-serif)
- **Taille** : 1rem (mobile) / 1.125rem (desktop)
- **Poids** : 400 (regular)
- **Hauteur de ligne** : 1.75 (pour une lecture confortable)
- **Couleur** : Gris foncé (#333333)

### Liens
- **Police** : Raleway (sans-serif)
- **Couleur** : Noir (#000000)
- **Décoration** : Souligné au survol uniquement
- **Transition** : Douce (0.2s)

## Espacement

### Entre les sections
- **Espacement vertical** : 2.5rem entre chaque section principale
- **Espacement horizontal** : Aucun (texte pleine largeur dans le container)

### Entre les paragraphes
- **Marge inférieure** : 1.5rem

### Listes
- **Espacement entre items** : 1rem
- **Marge gauche** : 1.5rem
- **Style** : Puces discrètes ou numérotation

## Éléments visuels

### Séparateurs
- **Ligne horizontale** : 1px, couleur #E5E5E5
- **Marge** : 3rem avant et après
- **Utilisation** : Entre sections importantes uniquement

### Mise en évidence
- **Texte important** : Gras (font-weight: 600)
- **Pas d'italique** : Garder le style minimal
- **Pas de couleurs** : Noir et gris uniquement

## Responsive

### Mobile (< 768px)
- **Padding** : Réduit à 2rem
- **Taille de police** : Légèrement réduite
- **Espacement** : Légèrement réduit

### Desktop (≥ 768px)
- **Largeur container** : 800px maximum
- **Padding** : 4rem horizontal, 6rem vertical
- **Espacement** : Confortable

## Navigation

### Fil d'Ariane (optionnel)
- **Position** : En haut de la page
- **Style** : Minimal, petit texte
- **Séparateur** : "/" ou ">"

### Retour
- **Lien** : "← Retour" en haut à gauche
- **Style** : Discret, Raleway, 0.875rem

## Footer de page

### Informations
- **Texte** : "Dernière mise à jour : [date]"
- **Style** : Petit, gris clair (#666666)
- **Position** : En bas de la page, centré

## Accessibilité

- **Contraste** : Minimum 4.5:1 pour le texte
- **Focus** : Visible sur les liens (outline)
- **Structure** : Hiérarchie H1 → H2 → H3 respectée
- **Lisibilité** : Taille de police minimum 16px

## Performance

- **Pas d'images lourdes** : Texte uniquement
- **Pas d'animations** : Transitions minimales
- **Chargement rapide** : Contenu statique

## Notes de design

- **Simplicité** : Moins c'est plus
- **Espace blanc** : Abondant pour aérer
- **Hiérarchie** : Claire grâce à la typographie
- **Cohérence** : Même style sur toutes les pages légales
